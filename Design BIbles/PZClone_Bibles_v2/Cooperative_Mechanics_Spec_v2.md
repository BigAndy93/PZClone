# PZClone — Cooperative Mechanics Spec v2

## Downed State
- Player incapacitated
- Teammate revive window
- Bleed-out timer

## Drag / Carry
- Movement speed penalty
- Noise increase while dragging

## Heavy Object Carry
- Requires 2 players
- Shared stamina drain

## Group Dynamics
- Shared noise consequences
- Shared risk in combat
