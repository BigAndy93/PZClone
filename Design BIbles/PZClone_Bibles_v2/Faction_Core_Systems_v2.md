# PZClone — Faction Core Systems Spec v2

## Faction Identity Model
Each faction contains:
- faction_id
- leader_id
- personality_profile
- aggression_level
- trust_baseline
- resource_pressure
- territory_zones

## Personality Profiles
- Defensive
- Opportunistic
- Isolationist
- Cooperative
- Militarized

Profiles influence diplomacy and raid likelihood.
